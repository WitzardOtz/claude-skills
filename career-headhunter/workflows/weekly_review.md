# Workflow — Weekly Review

## Trigger

Use this workflow for a weekly job-search recap.

## Steps

1. Read `07_application_tracker.csv`.
2. Read `03_job_evaluations.md` and `04_job_shortlist.md`.
3. Summarize:
   - roles found;
   - roles shortlisted;
   - applications submitted;
   - recruiter messages sent;
   - responses received;
   - recurring fit patterns;
   - recurring gaps;
   - search adjustments.
4. Update `08_learning_log.md`.
5. Recommend next week's priorities.
